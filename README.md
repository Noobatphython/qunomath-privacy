# QunoMath Privacy Policy

Privacy policy page for QunoMath, hosted via GitHub Pages.

## What This Is

This repository hosts the privacy policy for the QunoMath mobile app (available on Google Play). The page is served at:

https://noobatphython.github.io/qunomath-privacy/

## Tech Stack

- Static HTML
- GitHub Pages hosting

## Setup

No build step required. The `index.html` file is served directly by GitHub Pages.

To update the privacy policy, edit `index.html` and push to the `master` branch.
